# CasperJs Samples

This repository includes a simple webscrape script and a browser test suite using casperJS.

####  Installation
```bash
npm install -g phantomjs
npm install -g casperjs
```

####  To run the webscrape
```bash
casperjs scrape.js
```

####  To run the browser test
```bash
casperjs test test.js
```
